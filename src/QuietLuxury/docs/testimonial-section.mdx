# Testimonial Section

This section is a collection of testimonials.

## Preview

<Gallery images={['carousel.png', 'carousel-no.png']} />

## Main element

### Accepted elements before the header

-   Banner Image _(optional)_
    -   Takes an image element with size banner.
    -   The image will be displayed as the background if there are more than three testimonials.

### Accepted elements in the header

-   Title
-   Subtitle

## Item elements

### Accepted elements before the header

-   Company Image _(optional)_
    -   Takes an image element.
    -   The image will be displayed above the testimonial

### Accepted elements in the header

    - Subtitle
    - Description
      - Description is used as name of the tetimonial author.
