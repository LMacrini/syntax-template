# Page Header

## Main element

### Accepted elements before the header

  - Logo *(optional)*
    - Image component.
    - Will be displayed to the left of the navigation bar.

## Parameters

name | type | description
--- | --- | ---
`alignment` | String (`left` or `center`) | The alignment of the navigation bar.
`sticky` | Boolean | Whether the navigation bar should be sticky or not.
