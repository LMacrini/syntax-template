# Enterprise Solution Section

This section is a comparison between two solutions using bullet points. these could be two different products, or two different plans.

## Main element

### Accepted elements before the header

  - Background Image *(optional)*
    - Takes an image element with size banner.

### Accepted elements in the header
  - Subheading
  - Title
  - Subtitle

## Item elements (This component takes two)

### Accepted elements before the header
  - Icon
    - Takes an icon element.
    - The icon will be displayed above the title.

### Accepted elements in the header
  - Subtitle

### Accepted elements in the body
  - List Item
    - The component displays it as a bullet point.
