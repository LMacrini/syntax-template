# Feature Section

This section is a collection of features.

## Main element

### Accepted elements in the header

  - Subheading
  - Title
  - Subtitle

### Accepted elements in the body

  - Link
    - Takes a link element.
    - The link will be displayed as a button.

## Item elements

### Accepted elements before the header

  - Icon
    - Takes an icon element.
    - The icon will be displayed above the title.

### Accepted elements in the header

  - Subtitle
  - Description

### Accepted elements in the body

  - Link

## Customization

The alignment of the main header will change the layout of the page to one of three permutations.
